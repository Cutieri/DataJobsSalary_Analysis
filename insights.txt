Foi utilizada a variável Salario_em_USD para padronização, evitando distorções causadas por diferentes moedas.
Foi realizado um processo de feature engineering, agrupando cargos em categorias mais amplas de Área de Atuação:
Cientista de Dados
Engenheiro de Dados
Analista de Dados
Arquiteto de Dados
Gerente de Dados
Valores extremos (outliers) foram identificados utilizando o método IQR:
Q1: 62.726
Mediana: 101.570
Q3: 150.000
Limite superior: ~280.911
Os outliers não foram removidos, sendo considerados como salários esporádicos (ex: cargos executivos ou contextos específicos), mas reconhecidos como potenciais distorções em métricas agregadas.
