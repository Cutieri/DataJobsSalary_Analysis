## 🧠 Contexto

Este projeto analisa dados de salários na área de dados com o objetivo de entender padrões de remuneração e identificar fatores que impactam os salários.

A análise considera variáveis como experiência, área de atuação, localização e tipo de trabalho, buscando gerar insights relevantes sobre o mercado.

---

## 🎯 Objetivo

Gerar insights a partir dos dados para apoiar análises relacionadas a:

- Evolução salarial ao longo do tempo  
- Impacto da experiência na remuneração  
- Diferenças entre áreas de atuação  
- Influência do trabalho remoto  
- Distribuição geográfica dos salários  

---

## 📌 Principais Métricas

- 💰 **Salário Médio:** 112.297 USD  
- 📊 **Mediana Salarial:** 101.570 USD  
- 📉 **Desvio Padrão:** 70.957 USD  

---

## 🔍 Principais Insights

- 📈 **Crescimento do mercado ao longo do tempo**  
  → Salários aumentaram significativamente entre 2020 e 2022  

- 🎯 **Experiência impacta diretamente o salário**  
  → Profissionais seniores ganham mais que o dobro de juniores  

- 🏠 **Trabalho remoto apresenta maior remuneração**  
  → Média salarial superior em comparação ao presencial e híbrido  

- 🧠 **Funções estratégicas são mais valorizadas**  
  → Arquitetos e Gerentes de Dados possuem os maiores salários  

- 🌍 **Concentração geográfica relevante**  
  → Estados Unidos concentram maior volume de dados e salários elevados  

- ⚠️ **Presença de outliers salariais**  
  → Valores extremos impactam métricas como média e desvio padrão
